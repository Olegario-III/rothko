## Description

This project is an interactive **Rothko Painting Editor** built with vanilla HTML, CSS, and JavaScript. It allows users to recreate the signature style of abstract expressionist painter **Mark Rothko** by customizing the colors and sizes of stacked rectangular color fields on a framed canvas.

It completes the freeCodeCamp "Build a Rothko Painting" project in the Responsive Web Design certification.

### Features
- Framed canvas with three overlapping rectangular color blocks
- Control panel to select and modify:
  - Frame width
  - Canvas background color
  - Colors and vertical sizes/positions of the three color rectangles
- Real-time updates as you adjust values
- Reset button to restore default Rothko-inspired appearance
- Clean, centered layout
